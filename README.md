# aes-encryption-project

Project by Aleksy

Unless this repo receives an update which modifies how the program interacts with files, the user should follow this procedure when using it.

How to use:
  insert any .txt files into the /data/ directory and type their name out when you desire to use them in the program
  I have not checked but you should be able to put files in while the program is running and use them as normal.
  
  encrypted text files should begin with "enc_ "... at the beginning of their file name
  decrypted text files don't matter but they will start off with "dec_" ... and what ever filename the user gave the program
  plaintext files are plaintext, nothing special required.
